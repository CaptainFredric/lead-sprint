# Operator Dashboard

The weekly dashboard exists to prevent fake progress. A Lead Sprint operator can spend all week improving the site, tweaking copy, or inventing new offers. The dashboard keeps attention on the few numbers that actually move money.

## Weekly Inputs

Track:

- Weekly revenue target
- Outreach sent
- Replies received
- Reviews booked
- Pilots sold
- Care plans sold
- Cash collected
- Current blocker

Save the week locally in the browser at the end of each work block. Reset it when a new week starts.

## Score Meaning

The execution score rewards:

- Prospecting volume
- Reply signal
- Booked teardown reviews
- Closed pilots or care plans
- Cash collected against the weekly target

The score is not a moral grade. It is a focus tool. If the score is low, the next action should be simpler, not more creative.

## Operating Rules

- If outreach is low, send more focused teardown notes.
- If replies are weak, improve the first-line observation.
- If calls are low, ask warm replies for a 15-minute review.
- If calls are not closing, use the ROI calculator and mini-proposal.
- If pilots are sold but care is not, use the handoff report to pitch care.
- If cash is below target, collect before calling the week done.

## Weekly Review

Copy the generated weekly review every Friday or after a serious work block. Use it to decide one bottleneck for the next week.

Do not change the offer until the same bottleneck has been worked deliberately for at least one focused block.
